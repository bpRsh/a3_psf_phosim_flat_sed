a4_run_phosim_all_catalogs¶

Depends:
  • instance_catalogs/narrowband*.icat
  • narrowband_seds/narrowband*.sed
  • backgrounds/background1.bkg
Outputs:
  • phosim_output_zipped/narrowband*/17_zipped_psf_fitsfiles
Runtime:

Time taken: 2 days, 7 hours, 21 minutes, 32.907248 seconds. Machine : Macos Mavericks pisces 8 GB RAM Exptime : SIM_VISTIME 300 SED : original_seds/exp9_pf_12gyr_interpolated.cat and its childeren.

Note

  1. The program a1_create_background.py will create backgrounds/background1.bkg

2. The program a2_create_seds.py will create narrowband_seds/narrowband*.sed and narrowband_seds/broadband.sed.

3. The program a3_create_instance_catalogs.py will create instance_catalogs/narrowband*.icat and instance_catalogs/broadband.icat.

  1. We can change seed, magnitude , sed etc while creating instance catalogs.
  2. For 21 input instance catalogs this program will create 21 unzipped psfs.
  3. We need only electron image and for that we will use a5_unzip_all_psf.py.

Warning

To run phosim, we should use python2 not the ananconda python3. export PATH=”/Library/Frameworks/Python.framework/Versions/2.7/bin:${PATH}”
