a8_weighted_normalized_psf¶

This program creates weighted average of the input fitsfiles.

Inputs:

phosim_normalized_psf/psf*.fits
Outputs:

weighted_psf.fits
Info:
  1. This program creates weighted average of the input fitsfiles. Algorithm: output = data_i * weight_i / sum(weight)
  2. The weight range 1 to 1.2 are the average flux ratio of 100 f606 and f814 galaxies. (refer: ~/phosim/simdatabase/average_flux_ratio.py)
Runtime:

3 sec
